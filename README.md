# Design-and-Verification-ofFPGA-Based-Wireless-Control-and-Monitoring-System-Using-WiFi
Design and verify an FPGA-based wireless control and monitoring system using WiFi. Controls 8 LEDs and monitors sensors via UART commands over WiFi. Implemented in Verilog with comprehensive simulation testbench, enabling real-time remote device management and sensor feedback. Ideal for IoT applications.
